# TheAngel Website

Static site recreation of TheAngel.app, migrated from WordPress to GitHub Pages.

## Project Details

See `claude.md` for full project specifications, design guidelines, and development instructions.

## Quick Start

1. Clone the repository
2. Open `index.html` in a browser (or serve with a local server)
3. Deploy to GitHub Pages by pushing to the main branch

## Structure

- `index.html` - Main homepage
- `css/` - Stylesheets (styles.css, responsive.css)
- `js/` - JavaScript modules (main.js, trading-table.js, modals.js)
- `assets/` - Images and icons

## Deployment

Push to GitHub Pages. Site will be available at `https://<username>.github.io/theangel/`
